## Mortgage Calculator
This is simple application that implements mortgage calculator.

### Prerequisites
* Java 16
* Node 10.x
* Postman
* PostgreSQL
* Docker

### Technical Stack
* backend
    * Java 16
    * Spring Boot
    * Maven
    * PostgreSQL
* frontend
    * React

### Modules description
* `mortgage-calculator-server` backend with Spring Boot
* `postman-collections` Postman collections
    * MortgageCalculator.postman_collection.json
